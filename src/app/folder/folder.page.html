PROYECTOS



Proyecto 1: Inventario para tiendas

. El proyecto es sobre un sistema de información para poder: registrar productos con nombre, descripción, fecha de vencimiento, cantidad, además realizar ventas y llevar el control del inventario principalmente en tienda de barrio en Bosa, está diseñado para ser fácil de utilizar ya que tuvieron en cuenta que la mayoría de las personas que tienen este tipo de empresas son personas mayores, cuenta con varios módulos desde el Loguin hasta que se realiza la venta.

Proyecto 2: Inventario empresa de telas

El proyecto es sobre un sistema de información para poder: registrar productos con nombre, descripción, cantidad, además realizar ventas y llevar el control del inventario para una empresa de telas especifica, cuenta con varios módulos desde el Loguin hasta que se realiza la venta, lo que me gusto de este proyecto es que tiene una ventana donde muestra el producto más vendido y además que al realizar la compra indica si se tiene o no el inventario que se requiere disponible, ellos identificaron que este proceso se estaba realizando de forma manual y de ahí nace la idea del sistema de información.

Proyecto7: Ordenar en restaurante

El proyecto es sobre un sistema de información que permite ordenar comida para el restaurante, los integrantes del grupo notaron una necesidad ya que esto se estaba haciendo de forma manual, escribiendo en un papel lo que se quería comprar y así nació la idea de crearlo, cuenta con varios módulos y perfiles de acuerdo con el rol de cada persona.
